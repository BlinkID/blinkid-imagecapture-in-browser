BlinkID ImageCapture UI: examples

The goal of this page is to display minimal example and features of BlinkID ImageCapture UI component.

Important: to use BlinkID ImageCapture UI component it's required to have a valid license key. It's possible to obtain a free trial license key by registering to Microblink dashboard.

Minimal example

UI component with default behaviour where it's required to set valid license key, engine location and recognizer.

Keep in mind that this example uses additional code which sends an image to web API for processing. Complete code can be seen in examples/ui directory of this repository.

Recognizer: BlinkIdImageCaptureRecognizer

Note: since this page has multiple components, only this one has enabled drag and drop feature.

<blinkid-imagecapture-in-browser
  engine-location="http://localhost/resources/"
  license-key="LICENSE-KEY-GOES-HERE"
  recognizers="BlinkIdImageCaptureRecognizer"
></blinkid-imagecapture-in-browser>

Scan both sides

UI component with default behaviour where it's required to set valid license key and recognizer.

Keep in mind that this example uses additional code which sends an image to web API for processing. Complete code can be seen in examples/ui directory of this repository.

Recognizer: BlinkIdImageCaptureRecognizer

<blinkid-imagecapture-in-browser
engine-location="http://localhost/resources/"
license-key="LICENSE-KEY-GOES-HERE"
recognizers="BlinkIdImageCaptureRecognizer"
recognizer-options="captureBothDocumentSides"
></blinkid-imagecapture-in-browser>

Custom messages

It's possible to change default text messages which are displayed to the user.

Use attribute/property translations to change default messages.

Keep in mind that this example uses additional code which sends an image to web API for processing. Complete code can be seen in examples/ui directory of this repository.

Recognizer: BlinkIdImageCaptureRecognizer

<blinkid-imagecapture-in-browser
  engine-location="http://localhost/resources/"
  license-key="LICENSE-KEY-GOES-HERE"
  recognizers="BlinkIdImageCaptureRecognizer"
></blinkid-imagecapture-in-browser>
<script>
  const el = document.querySelector('blinkid-imagecapture-in-browser');

  // See src/utils/translations.ts for possible translation keys
  el.translations = {
    'action-message': 'Alternative CTA'
  };
</script>
